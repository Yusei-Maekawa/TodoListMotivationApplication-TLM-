# mokuvation

## 概要
TodoListMotivationApplication は、短期・中期・長期の目標を整理し、それぞれを達成するための行動を記録・可視化することで、継続的にモチベーションを保ちながら目標達成を支援するアプリケーションです。

単なる ToDo 管理ではなく、目標を段階的に分解し、日々の行動や達成状況を振り返りやすくすることで、「続けられること」に重きを置いた設計を目指しています。

---

## 開発背景
日常生活の中では、短期・中期・長期といった複数の目標を同時に持つことが多い一方で、目標が曖昧なままになったり、途中で進捗が見えなくなったりして、モチベーションを維持できずに止まってしまうことがあります。

そこで本アプリでは、目標を整理・分解し、達成までの流れを見える化することで、利用者が自分の行動を振り返りながら継続しやすい仕組みを提供することを目的としています。

---

## 目的
- 短期・中期・長期の目標を一元管理できるようにする
- 目標を具体的な行動に落とし込みやすくする
- 達成状況や未達成状況をわかりやすく可視化する
- いつ・何をしたかを記録し、振り返りやすくする
- モチベーションを維持しやすい体験を提供する

---

## 想定するユーザー
- 学習や資格取得に向けて継続的に努力したい人
- 日々の目標管理を可視化して達成感を得たい人
- 長期的な目標に対して、今やるべきことを整理したい人
- タスク管理だけでなく、行動記録や振り返りも行いたい人

- ### ペルソナ例1：仕事の目標を管理したい会社員
- 28歳・会社員
- 業務改善や資格取得、キャリアアップに向けた目標を持っている
- 毎日の仕事に追われ、中長期の目標が後回しになりやすい
- 短期・中期・長期の目標を整理し、進捗を見える化しながら継続したい
- 小さな達成を積み重ねて、前向きに取り組める環境を求めている

### ペルソナ例2：生活習慣を改善したい社会人
- 30歳・社会人
- 運動、睡眠、食事改善など、日々の生活習慣を整えたいと考えている
- 目標を立てても、忙しさや気分に左右されて継続できないことが多い
- 毎日の行動記録や達成状況を見える化し、継続のモチベーションにつなげたい
- 将来的には健康的な生活を習慣化したいと考えている

### ペルソナ例3：個人開発や副業を進めたい人
- 25歳・社会人
- 本業と並行して個人開発や副業に取り組みたいと考えている
- やりたいことは多いが、優先順位が曖昧になり進みが止まりやすい
- 長期目標から逆算して、今やるべきことを明確にしたい
- 作業履歴や進捗を可視化し、継続しやすい仕組みを求めている

### ペルソナ例4：趣味や創作活動を継続したい人
- 24歳・社会人
- イラスト、動画制作、配信、楽器練習などの趣味を継続したい
- モチベーションの波があり、続けられる時と止まる時の差が大きい
- 自分の取り組みを記録し、成長や継続を実感できるようにしたい
- 楽しさを保ちながら、目標に向かって少しずつ進めたい

## MVPで実装したい機能
### 1. 目標管理機能
- 短期・中期・長期の目標を登録
- それぞれを1セットとして管理
- 登録した目標を後から見返せる

### 2. タスク管理機能
- 各目標に紐づく ToDo を追加
- タスクの達成 / 未達成を管理
- 進捗状況を確認できる

### 3. 可視化機能
- 達成・未達成・進行中が一目で分かるUI
- 目標全体に対する進捗率の表示
- 視覚的に状況を把握しやすいデザイン

### 4. 記録・振り返り機能
- いつ何を行ったかを記録
- 過去の行動履歴を確認
- 継続状況や努力の流れを振り返る

### 5. スケジュール機能
- タスクや目標に期限を設定
- 日付ベースで行動履歴を確認
- スケジュール感を持って目標達成を目指せる

## このアプリの特徴
本アプリの特徴は、単なる ToDo 管理ではなく、**目標の整理・行動の記録・達成状況の可視化・継続支援** をまとめて行える点にあります。

特に、短期・中期・長期の目標をつなげて管理することで、「今やっていることが将来のどの目標につながるのか」を意識しやすくし、継続のしやすさを高めることを目指しています。

## 画面イメージ
### ホーム画面
- 現在の目標一覧
- 今日やること
- 進捗状況の表示

### 目標詳細画面
- 短期・中期・長期目標の確認
- 紐づくタスク一覧
- 達成状況の確認

### 記録画面
- 今日やったことの入力
- 行動内容やメモの保存

### 履歴 / カレンダー画面
- 日ごとの行動記録の確認
- 過去の達成履歴の振り返り

## 将来的な展望
今後は、利用者ごとに使いやすい目標管理方法を選べるようにし、マンダラチャート形式や分析機能なども取り入れながら、より継続しやすく、前向きに行動できるアプリへ発展させていきたいと考えています。

最終的には、「目標を立てる」だけで終わらず、「行動し続けられる」「振り返って次につなげられる」体験を提供できるアプリを目指します。
